## Dependancies

To be able to use the setting you need to have a media server with <a href=https://github.com/bluenviron/mediamtx>Mediamtx</a>.

Also, you will need yarn for managing npm packages: <a href=https://yarnpkg.com/getting-started/install>Yarn</a>

For devloppement you will need to satisfy the  dependancies for Tauri for your Os please refer to their documentation:  <a href=https://tauri.app/v1/guides/getting-started/prerequisites>Tauri prerequisites</a>

After installing to run a development version of the app run:
```
yarn tauri dev
```

## Installation
### Windows
For Windows, you have a .msi and a .exe  you can download in the release section of the GitHub repository. 

### Ubuntu and Debian based linux distribution

you can use the .deb and install it with apt.

### Other Distribution

There is an Appimage you can Download.
