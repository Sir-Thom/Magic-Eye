# Magic Eye

</p>
<p>This small gui application let you stream video to a other computer with a webcam or the raspberry pi cam. </p>


## Table of Contents

- [Installation](#installation)
- [Uninstall](#uninstall)
- [Update](#update)

## Installation
```
git clone https://github.com/Thomas-Toulouse/Magic-Eye.git
cd Magic-Eye
make
```
## Uninstall
```
cd Magic-Eye
make uninstall
```
## Update
```
cd Magic-Eye
make update
```
